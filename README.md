# Universal Overbought Calculator

Готовый проект для GitHub/Vercel.

## Что умеет
- загрузка `.xlsx` / `.xls`
- формат `DateTime + SPX`, `Date + Gold`, `Date + BTC` и любые аналогичные таблицы
- поддержка нескольких листов Excel
- автоопределение колонки даты
- автоопределение числовых колонок активов
- ручной выбор нужного актива из списка колонок
- расчёт `MA200`, `RSI(14)`, `Deviation %`, `Signal`
- график `Price vs MA200`
- график `История перекупленности`

## Запуск локально
```bash
npm install
npm run dev
```

## Сборка
```bash
npm run build
```

## Деплой на Vercel
1. Загрузите проект в GitHub.
2. Импортируйте репозиторий в Vercel.
3. Нажмите Deploy.

## Подходящий формат Excel
Пример:

| DateTime | SPX |
|---|---:|
| 1929-12-31 | 100 |
| 1930-01-02 | 98.74 |

Или несколько активов:

| Date | SPX | Gold | BTC |
|---|---:|---:|---:|
| 2024-01-01 | 4800 | 2050 | 42000 |

После загрузки можно выбрать нужный столбец актива в интерфейсе.

# Overbought Calculator

Калькулятор перекупленности для любых активов из Excel.

## Возможности
- Загрузка Excel `.xlsx/.xls`
- Автоопределение столбца даты и цены
- Расчёт `MA200`
- Расчёт `Deviation %` от `MA200`
- Рекомендация `BUY / HOLD / SELL`
- График цены и `MA200`

## Запуск
```bash
npm install
npm run dev
```

## Деплой на Vercel
Просто импортируйте репозиторий в Vercel.
Важно: если обновляете старый проект, удалите старый `package-lock.json`.
